给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

**示例 1：**

```bash
输入：s = "()"
输出：true
```

**示例 2：**

```bash
输入：s = "()[]{}"
输出：true
```

**示例 3：**

```bash
输入：s = "(]"
输出：false
```

**示例 4：**

```bash
输入：s = "([)]"
输出：false
```

**示例 5：**

```bash
输入：s = "{[]}"
输出：true
```

**提示：**

- `1 <= s.length <= 104`
- `s` 仅由括号 `'()[]{}'` 组成

***

**解答:**

```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
var isValid = function(s) {
    while(s.includes('[]')||s.includes('()')||s.includes('{}')){
        s = s.replaceAll('[]','').replaceAll('()','').replaceAll('{}','')
    }
    return s===''
};
```

**思路:**

如果括号成对的,那么替换括号以后再次替换,如果还存在成对的就说明还可以继续替换下去,直到无法找到成对的括号。最后比较是否是空字符串,如果是就说明这个字符串括号位置都正确,否则就不正确。
